# ObliqueCutImageView
a ladder-like ImageView
一个可以把图片切割成梯形的ImageView

### 主要参数(param)
 - direction:切割起点角
   * LeftTop 
   * LeftBottom 
   * RightTop 
   * RightBottom
 - orientation:切割方向
   * vertical
   * horizantal
 - extent:切割距离(length)
   * VERTICAL : extent <= imageView.width
   * HORIZANTAL : extent <= imageView.height
